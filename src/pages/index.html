ESP32 & M5Stack Friendly

Deploy, manage and explore Launcher firmware with confidence.

Launcher streamlines installation workflows for a growing catalog of ESP32-based devices including M5Stack, LilyGO and custom boards. The project combines an accessible web flasher, curated firmware builds and clear documentation so that makers and educators can ship ideas instantly.
